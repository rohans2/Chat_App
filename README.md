# Chat_App
An app made using firebase, having all the modern-day chat app features including - <br>
->User Profiles<br>
->Read - Delivered notifiers<br>
->Notifications<br>
->Online - Offline Status of Users<br>
->Edit User Display Picture Options.<br>
